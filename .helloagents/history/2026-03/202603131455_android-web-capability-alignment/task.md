### 1. 认证与会话生命周期对齐

- [√] 1.1 统一 Android token 校验失败后的清理逻辑（清 token / 清当前身份 / 回登录）
  - 验证: 无效 token 启动时不会停留在半登录态

- [√] 1.2 统一 Android forceout / reject 处理为会话生命周期事件

- [√] 1.3 统一 Android logout 行为：清 token、清 current session、断开 WS、重置相关状态
  - 验证: 退出后无法继续访问会话和聊天页

- [√] 1.4 校正 `LiaoApp` 的启动/恢复导航逻辑，支持基于 token + current session 的恢复

### 3. 会话列表状态层对齐
